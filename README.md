Это backend-приложение для управления и отслеживания вакансий и откликов.
Проект помогает структурировать процесс поиска работы: хранить вакансии, отслеживать статусы, управлять заметками и аналитикой.

Основные возможности
📄 CRUD операции для вакансий
🏢 Хранение информации о компании
📊 Отслеживание статусов (Applied, Interview, Offer, Rejected и т.д.)
🗓 Управление датами (отклик, интервью и др.)
🔎 Фильтрация и поиск вакансий
🧩 REST API для интеграции с frontend / Postman
🐳 Поддержка Docker (если у тебя есть compose — оставляем)

Технологии:
-Java 17+ -Spring Boot -Spring Web -Spring Data JPA -Hibernate -PostgreSQL / H2 -Docker (опционально) -Gradle
